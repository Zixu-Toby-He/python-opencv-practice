# OpenCV 练习测试

本项目是一系列 OpenCV 基础功能的 Python 练习脚本，涵盖了图像读取与显示、色彩空间转换、通道操作、几何变换、傅里叶变换、视频处理、摄像头调用以及边缘检测等常见计算机视觉任务。所有代码均以中文变量和注释编写，适合初学者对照学习。

## 环境依赖

- Python 3.6+
- OpenCV (`opencv-python`)
- NumPy
- Pillow (PIL) —— 用于绘制中文文字
- 标准库：`os`, `pathlib`, `random`, `traceback`, `time`, `bisect`

安装依赖（推荐使用虚拟环境）：

```bash
pip install opencv-python numpy pillow
```

> **注意**：`简单图像绘制.py` 中使用了 Windows 下的中文字体 `C:/Windows/Fonts/simsun.ttc`，若在其他系统运行，请替换为相应的中文字体路径或删除中文绘制部分。

## 各模块功能说明

### 1. 图片输入输出与显示 (`图片输入输出与显示.py`)
- 从 `输入图片/` 随机选择一张图片
- 支持中文路径读取（使用 `np.fromfile` + `cv2.imdecode`）
- 显示彩色/灰度图像
- 将处理后的图片保存到 `输出图片/` 目录

### 2. 色彩空间转化 (`色彩空间转化.py`)
- 读取第一张图片，展示 BGR 原图
- 转换为 Gray 灰度图和 HSV 色彩空间并显示

### 3. 色彩通道 (`色彩通道.py`)
- 随机选图，分离 B、G、R 通道并分别显示
- 合并通道演示颜色混合
- 分离 HSV 通道，展示色调(H)、饱和度(S)、亮度(V)对图像的影响
- 通过滑块式修改展示各通道值变化效果

### 4. 简单图像绘制 (`简单图像绘制.py`)
- 创建空白画布 (1920×1080)
- 绘制彩色矩形块、线段、矩形框、圆形、多边形
- 使用 OpenCV 绘制英文文字
- 借助 Pillow 绘制中文文字（需要系统字体支持）

### 5. 几何变换 (`几何变换.py`)
- 随机选图并缩放（`resize` 通过 `dsize` 或 `fx/fy`）
- 图像翻转（x轴、y轴、中心对称）
- 仿射变换：平移、旋转（利用 `getRotationMatrix2D`）、倾斜（自定义仿射矩阵）
- 透视变换（`getPerspectiveTransform` + `warpPerspective`）

### 6. 傅里叶变换 (`傅里叶变换.py`)
- 对 BGR 三通道分别进行离散傅里叶变换 (`cv2.dft`)
- 使用 `np.fft.fftshift` 将低频移到中心
- 显示各通道的频域实部图像
- （代码底部注释了一段频域滤波示例）

### 7. 傅里叶加水印 (`傅里叶加水印.py`)
- 实验性质：读取图片，对三个颜色通道进行 DFT
- 代码中注释了生成密文板（文字水印）和在频域嵌入水印的部分
- 目前仅展示了 DFT -> 直接 IDFT 还原的图像（未实际嵌入水印）

### 8. 视频绘制 (`视频绘制.py`)
- 随机选图，生成一个旋转动画视频（mp4格式）
- 可设置帧率、时长，使用 `cv2.VideoWriter` 写出
- 生成的视频保存在 `输出图片/` 目录

### 9. 读取摄像头 (`读取摄像头.py`)
- 遍历摄像头索引（0~3），打开可用的摄像头
- 实时显示多路摄像头画面（如果有多个设备）
- 按 `q` 键退出，释放资源

### 10. 边缘检测 (`边缘检测.py`)
- 固定读取 `8.png`，将其转为灰度图并进行中值滤波
- 使用 Canny 算子进行边缘检测
- 使用霍夫圆变换 (`cv2.HoughCircles`) 检测圆形
- 在局部区域二次验证圆形，并绘制检测结果

### 11. 独立脚本

#### `opencv练习.py`
- 项目的主入口，通过注释/取消注释来快速调用 `分测试` 中的各个模块。

#### `生成圆形动画.py`
- 根据自定义的圆形运动轨迹（包含时间、位置、半径、颜色）生成动画视频。
- 支持多个圆同时运动，使用线性插值平滑位置变化。

#### `随机生成圆.py`
- 批量生成包含随机颜色、大小、位置圆形的图片，可用于测试素材生成。

## 使用方法

1. 将测试图片放入 `输入图片/` 文件夹。
2. 运行根目录下的 `opencv练习.py`（取消对应模块的注释即可运行）。
3. 也可以直接运行 `分测试/` 中的各个脚本，它们会自动从 `输入图片/` 读取图片并将结果保存到 `输出图片/`。
4. 执行过程中会弹出 OpenCV 窗口显示处理结果，按任意键继续。
5. 独立脚本可直接运行，例如：
   ```bash
   python 生成圆形动画.py   # 会生成 视频.mp4
   python 随机生成圆.py     # 会在当前目录生成 10~19.png
   ```

## 注意事项

- 图片路径支持中文，代码中统一使用 `np.fromfile` + `cv2.imdecode` 方式避免中文路径问题。
- 部分脚本会随机选择图片，因此每次运行结果可能不同。
- `简单图像绘制.py` 中的中文字体路径仅适用于 Windows 系统，Linux/macOS 需要手动修改。
- `傅里叶加水印.py` 仍处于实验阶段，水印嵌入和提取功能尚未完成。
- 摄像头读取脚本需要在有摄像头的设备上运行，且部分索引可能不可用。

## 示例输出

运行 `色彩通道.py` 后，`输出图片/` 目录会生成类似 `1-彩色.jpg`, `1-黑白.jpg` 等文件。  
运行 `视频绘制.py` 会生成 `xxx-rotate.mp4` 的旋转动画视频。  
运行 `边缘检测.py` 会弹出窗口展示检测到的圆形。
